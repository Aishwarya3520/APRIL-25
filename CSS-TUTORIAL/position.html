CSS position Property

The position property specifies the type of positioning method used for an element.

  • Static: Default value, elements are positioned according to the normal flow of the document page.
  • Relative: Elements are positioned relatives to their normal position.
  • Absolute: Elements are positioned relatives to the nearest postioned ancestor (insted of positioned relatives to the viewport like fixed).
  • Fixed: Elements are positioned relatives to the browser window.
  • Sticy: Elements are positioned based on the user's scroll position.
